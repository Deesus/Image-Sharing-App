This prototype was created by Dee Reddy. The goal was to create an app that emulates an image sharing site -- e.g. Flickr, Imgur, etc. The back-end runs on Flask -- a Python web-framework, and files and information are stored in a PostgreSQL database. Image uploads are handled by the app itself (as opposed to merely pointing to a link).
